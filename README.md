# Stormworks-Mods
Mods for Stormworks - Build and Rescue
For more info visit nexusmods.com/stormworksbuildandrescue

### Repository Contents
* Creative Generator
* Creative Battery
* Small Foghorn
* Small Hinge


## Installation:
```
Extract xml files to: \Steam\steamapps\common\Stormworks\rom\data\definitions
```
